# نشر لوحة الوحدة الأولى على GitHub Pages

هذه النسخة مخصصة لـ GitHub Pages، ولذلك تعمل باستخدام HTML وCSS وJavaScript فقط.

## ملاحظة مهمة
- يستطيع جميع الطلاب فتح الرابط وحل الأسئلة.
- تقدم كل طالب يحفظ داخل متصفح جهازه باستخدام Local Storage.
- النتائج لا تصل إلى المعلم في لوحة مركزية.
- فتح الرابط من جهاز أو متصفح آخر يبدأ تقدمًا جديدًا.

## خطوات النشر
1. أنشئ Repository جديدًا على GitHub واجعله Public.
2. ارفع محتويات هذا المجلد إلى جذر المستودع.
3. افتح Settings ثم Pages.
4. في Build and deployment اختر Deploy from a branch.
5. اختر الفرع main والمجلد /(root)، ثم Save.
6. انتظر عدة دقائق ثم اضغط Visit site.

الرابط عادة:
https://USERNAME.github.io/REPOSITORY-NAME/

## تحديث الأسئلة
عدّل ملف:
data/questions.json

ثم ارفع النسخة الجديدة إلى GitHub، وسيتم تحديث الموقع تلقائيًا بعد النشر.
